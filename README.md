# 📊 Super Store Sales Dashboard (Power BI)

## 📌 Project Overview
This project presents an interactive **Super Store Sales Dashboard** developed using **Microsoft Power BI**. The dashboard provides key business insights by analyzing sales, profit, orders, products, categories, regions, and customer segments. It enables users to monitor business performance through dynamic visualizations and interactive slicers.

---

## 🎯 Objectives
- Analyze overall business performance.
- Track Total Sales, Profit, Orders, and Quantity.
- Identify Top 10 Best-Selling Products.
- Compare Sales across Categories and Regions.
- Monitor Sales Trends over time.
- Enable interactive filtering using slicers.

---

## 📈 Dashboard Features

### KPI Cards
- 💰 Total Sales
- 📦 Total Orders
- 💵 Total Profit
- 📊 Total Quantity

### Visualizations
- Sales by Category (Column Chart)
- Sales by Region (Column Chart)
- Top 10 Products by Sales (Bar Chart)
- Sales Trend by Order Date (Line Chart)

### Interactive Slicers
- Category
- Region
- Segment
